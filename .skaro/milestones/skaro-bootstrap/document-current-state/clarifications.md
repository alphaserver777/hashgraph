# Уточнения

## Базовые допущения
1. Skaro внедряется вручную в уже существующий репозиторий.
2. Непосредственная цель состоит в понимании проекта и обеспечении непрерывности будущей работы LLM, а не в поставке новой функциональности.
3. Поведение существующего кода не должно меняться, если только небольшое документационное улучшение не дает явной пользы.
4. Русский язык допустим для проектных Skaro-артефактов, потому что в репозитории уже есть смешение русского и английского.
5. Созданные артефакты должны отражать код в состоянии на 2026-03-15.
